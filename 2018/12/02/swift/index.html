Swift - 雨燕（学习总结）

Swift - 雨燕（学习总结）

一.SWift基础

(一)Swift

  • 敏捷，快速
  • 新一代苹果平台（iOS/macOS/watchOS/tvOS）的APP及游戏开发语言。
  • 语法与JavaScript、C#等语法类似，上手极快。

(二)历史

  • 2014年六月推出第一个版本，有苹果员工Chris Lattern历经四年发明。
  • Swift已经开源，计划逐渐替代ObjectiveC，成为iOS开发的主流语言。
  • 长期致力于二十年内运行到每一部智能设备上。

(三)Swift的创新

  1. 于引入Optional（可选类型）。
  2. 支持传统的“面向对象编程”，提升至“面向协议编程”；即由”类“到”值“，提倡使用结构体取代类（class），”遵守协议“取代”继承“。

(四)Hello World

print(1) print("hello world")

(五)常量赋值

  • 给常量一个名字并赋予值，也就是定义了一个常量。
  • 形式：let 常量名字 = 值（value）

(六)变量

  • 可主动变更或者预期范围内不固定，比如加班时间、股票指数、驾照通过几率。
  • 定义方法： var 变量名 = 值

(七)代码注释

  • 有助于自己理解
  • 便于他人理解

  • 专业优雅

    let π = 3.14

    //定义了一个武汉地区的白天电费。

    let 电费 = 0.5

(八)如何正确使用常量与变量

  • 优先使用常量
  • 由编译器来决定（报错更改）

(九)值的类型和类型推断

  • 3，6，9这样的整数，在Swift中称为Int类型的值。
  • 形式：var 变量名: Int = 3 由于Swift有类型推断，类型可以省略不写

var run = 5 print(run)

(十)浮点型（小数）

*Swift 中默认浮点型十Double（双精度）

var price = 2.5

(十一)类型安全

  • 变量一旦定义，其类型不可更改，即：不能给变量一个类型不同的值。

(十二)布尔型：Bool，表示逻辑上的“真”或者“假”

  • 只有两个值，ture和false

(十三)元祖:定义变量的一个组合

  • 形式（一般）：（3,”天”,”Swift”,”4”）
  • 形式（前缀）：（day：3，unit：”天”，lang:”Swift”,ver:”4”）

(十四)可选类型（Optional）：代表变量可能有值的情况。

  • 如：用户资料的选填部分，如地址 性别等辅助信息。
  • 形式： var 变量：类型？，默认是无值（nil）
    var wenxinyu : String?

二.基础操作符

(一)操作符简介

  • 一个符号或单词，类似数学中的+ - * /

(二)术语

操作几个对象，被称之为几元操作符。

  • 操作一个目标，是一元操作符。
  • 二元操作符操作2个目标，在2个目标之间，前后用空格分隔。
    3 + 6

(三)赋值操作符

  • 赋予或更新一个变量

(四)数学操作符：+ — * / %

(五)组合赋值操作符：赋值与数学操作符的组合，eg.+ =

(六)比较操作符：> >= < <=.结果是true或false

1 > 2 false
3>=3 true
2<1 false
2<=5 true

(七)逻辑操作符：用于表示逻辑中的“与”、“或”、“非”

  • 与：每项全部为true，整体为true

true && true && true

  • 或：有一项为true，整体为true

true || false || false

  • 非：对Bool值取反

let a = false !a true

四.集合类型

一组同类型的值的组合，根据组合的整体特性分为：

  • 有序可重复 - 数组（Array）
  • 无需不重复 - Set
  • 无序可重复，但每个值有唯一的键（Key）- 字典（Dictionary）

批量处理集合中的元素，可以使用for in 循环

(一)数组：有序可重复，用于存储同一类型的值

  1. 创建一个有默认值的数组
    let array1 = Array(repeating: "小姐姐", count: 30)
  1. 创建一个整数区间内元素的数组,Array(区间)

let intRangeArray = Array(-10...10)

  1. 用数组字面量来创建数组: [值1, 值2, …, 值N]

var topLangs = >["Swift","Python","Kotlin","JavaScript"]

    topLangs.count
    topLangs.isEmpty

  • ②➕添加: append, 添加一个同类型数组 +=
    topLangs.append(“php”)
    let h5 = [“HTML”,”CSS”]
    topLangs += h5

  • ③获取元素 : 数组名[索引], ⚠️索引总是从0开始
    topLangs[5]

  • ④插入: insert
    topLangs.insert(“React Native”, at: 1)

  • ⑤移除: remove
    topLangs.remove(at: 1)

高阶操作①: 求和，连接元素， reduce

高阶操作②: 变形 map

  • 交集 intersection
  • 补集 symmetricDifference
  • 差集 subtract
  • 并集 union

  • 相等: 两个集合有完全相同的元素.
  • ①子集: isSubset(可以相等), 严格子集isStrictSubset
  • ②父集: isSupersetOf(可以相等), 严格父集

七.枚举（Enumeration）

  • 计数、列举（有限的情况）；详细叙述之意。“增强的、更抽象型”的数字类型。
  • 非常强大，与switch语句结合使用，天生一对。
  1. 定义和使用：

*给定一个名称，然后把每一种情况列举出来。

  • 使用：点语法

let todayWeather = Weather.snowy

  • 与switch配合使用。如果变量是枚举值，可省略枚举名。如果switch语句不列举枚举类型的所有情况，则必须要有default语句来处理其他情况。
  • 附加值：每一种类型都可附加一个或多个值，形式是元组。case类型（Int，String）
  • 赋予附加值：加上元组字面量。
  • 取得附加值：用switch语句，在case中加上元组变量。

八.面向对象

  • 面向过程式的编程，在1960年左右的大型程序时遇到不少瓶颈和问题。
  • 此后的高级语言，开始借鉴面向对象的思维方式，发展到今天更加完善。
  • 面向对象主要从生物的特征、行为、繁殖、代差几个方面来思考。
  • Swift中，类（class）、结构体（struct）、枚举（enum）都有面向对象功能。

比如一只可爱的小黄鸭，有毛色、会划水，长大之后毛色会变。小黄鸭祖类（野鸭），亲戚类（鸳鸯）。

九.错误处理

错误处理：反映运行出错的“细节”，并恢复程序的过程。

  • 可选类型（Optional）值处理值缺失的情况，错误处理可以针对不同的出错原因对不同的应对。
  1. 一个函数可以加上thorws关键字，表示可以处理错误，这个函数的调用者可以捕获（catch）这个错误并进行应对。

2.当你调用可以抛出错误的函数，需在前面加上try关键字。

  • 处理更细分的错误情况，错误类型须遵从Error协议。
  • 有时候仅关心结果有误，可使用 try？来忽略错误细节。
  • try！强制执行可能会出错的方法，如果出错，则程序崩溃。

十.协议

协议：方法、属性或一段功能的蓝本。

  • 任意类型都可以“领养”协议，从而实现蓝本；也称“遵从”该协议。
  • Swift的特色：面向协议编程，鼓励多“领养”协议来替代类继承、因为“领养”的代价元低于继承。

1.两个空协议

2.“领养/遵从”若干个协议，要用逗号隔开

struct c: a, b{}

3.如果同时有继承和遵从协议，协议写在父类之后。

(一)属性协议

属性协议：遵从这要实现协议规定的属性。

属性必须定义为变量。setter可选实现。

  • 遵从属性协议：必须要实现其规定的属性。

  • 把名字相关属性从student分离

(二)方法协议

方法协议：定义时没有花括号执行体，实现仅要求名称相同。

(三)多重继承

协议继承：一个协议可以继承若干个协议，并可以在继承基础上增加新需求。（面向协议编程）

(四)协议作为类型使用

协议作为类型使用：可用于参数类型/返回类型；变量/常量/属性；常量类型中的元素类型

十一.扩展

扩展：为既有类型添加新的功能，及时无权查看源代码（也叫：追加建模）

范围：扩展计算属性，方法（包括构造器、下标）；扩展嵌套类型；扩展协议。

十二.泛型

  • 泛型：“范”指通用、普遍。现代编程必不可少的组成部分。
  • 数组和字典都是“泛型”集合。可以使Int，String，或者任何其他任何类型，无任何类型的限制。
